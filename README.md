# 🐙 Octopus VPN Bot v2

## Установка

```bash
npm install
```

## Настройка

1. Скопируй `.env.example` в `.env`
2. Получи API_ID и API_HASH на https://my.telegram.org
3. Получи BOT_TOKEN от @BotFather
4. Авторизуй свой аккаунт:

```bash
npm run auth
```

Скопируй SESSION_STRING в `.env`

## Запуск

```bash
npm start
```

## Флоу бота

1. 🔑 Получить ключ
2. Выбор клиента (Outline / Happ)
3. Выбор страны
4. Выбор срока
5. Выбор тарифа (Классический / Максимальный)
6. Оплата (заглушка)
7. Автоматическое получение ключа от TON VPN
8. Конвертация для Happ если нужно
9. Выдача ключа клиенту
